The same task, before and after translation.

On this set the models read patient phrasing perfectly. The losses appear after the translation step.

The figures elsewhere on this site trace circuits inside one small model. This page asks a different question of production-scale models: can a Claude model extract the clinical concepts from a sentence, and does translating the sentence first help or hurt? Each model is scored on the same fixed set of items, so the two percentages are directly comparable.

What the percentages mean

Each evaluation item is a sentence in raw patient phrasing plus a list of expected clinical terms. The model is asked to name the clinical concepts in the sentence; the answer scores a hit only if it contains the item's expected terms. Every model sees the identical items.

  • Patient phrasing The share of items answered correctly when the model reads the patient's own words, untouched. 100% = every item correct.
  • Clinician phrasing The same model first translates each patient sentence into clinician wording, then the identical extraction task is re-scored on that translation, against the same expected terms.
  • Δ Clinician minus patient. Because the items and the scoring are identical in both stages, Δ isolates what the translation step itself added or destroyed. A negative Δ means the model's own translation discarded clinical content it had understood perfectly in the original.

Results

The reading is the reverse of the gemma story told by the circuit figures. gemma-2-2b, a 2-billion-parameter model, measurably loses the clinical continuation when it reads patient phrasing. These production models did not: on this set, every model scored 100% on the untouched patient wording. Where content was lost was the translation step itself — the same failure mode the engine flags as a translation regression, and the reason the Translation figure's mitigation cannot be assumed safe without checking its output.

Read this with its limits

The item set is small, so single items move the percentages in large steps; treat the numbers as a demonstration of the harness and the direction of the effect, not as a ranking of models. Scoring is term-matching against expected clinical terms: an answer phrased with a valid synonym the item list lacks would score as a miss. Runs hit a live API on a specific date and can differ on re-run. See the broader limitations for what this evidence does and does not show.
